# 🏢 El Molino - Sistema de Gestión de Reservas

Bienvenido al repositorio oficial de **El Molino**, una plataforma integral diseñada para la gestión eficiente de espacios comunes y reservas en conjuntos residenciales. Este proyecto combina una arquitectura de backend robusta con una interfaz de usuario moderna y de alto rendimiento.

## 🌟 Características Principales
- **Gestión de Residentes**: Autenticación segura y panel personalizado para reservas.
- **Control Administrativo**: CRUD completo de recursos (piscinas, salones, canchas) y monitoreo de estados.
- **Diseño Premium**: Interfaz basada en Glassmorphism con efectos de transparencia y desenfoque.
- **Arquitectura Desacoplada**: Backend en .NET y Frontend en Next.js comunicados mediante APIs REST.

## 📂 Estructura del Repositorio
El repositorio está organizado profesionalmente para separar responsabilidades:

```bash
.
├── Backend/          # Solución .NET Core (Clean Architecture)
├── Frontend/         # Aplicación Next.js + React
├── Docs/             # Manuales técnicos y de usuario
└── README.md         # Esta guía general
```

## 🛠️ Stack Tecnológico Global
- **Backend**: .NET 10, Entity Framework Core, SQLite, JWT Authentication.
- **Frontend**: Next.js 15+, React 19, Vanilla CSS (Custom UI System).
- **Herramientas**: Git para control de versiones.
